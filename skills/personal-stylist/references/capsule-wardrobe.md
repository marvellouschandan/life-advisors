# Minimal Capsule Wardrobe (base for 40+ combos)

Locked palette: Navy, Olive, Black, White, Beige, Charcoal, Maroon, Rust.
Keep everything deep, solid, structured. No loud prints or big logos.

1. Dark blue jeans (slim/straight)  2. Black jeans  3. Beige chinos
4. Olive chinos  5. Charcoal trousers  6. Black/olive cargos (tapered)

Polo (1): navy/olive
Kurtas (2): navy/charcoal short kurta, maroon/rust kurta
Layers (3): olive overshirt, denim jacket/overshirt, navy/grey blazer

## Footwear (6) — use letter codes in captions

## Combo logic (mix and match)
- T-shirt + chinos/jeans + A/D  (casual)
- Shirt + chinos/trousers + C/F  (smart casual → semi formal)
- Polo + chinos + A/C
- Overshirt/denim jacket over tee + jeans/cargos + A/B  (layered casual)
- Blazer + shirt + trousers + F  (formal)
- Kurta + jeans/chinos + D  (Indo-fusion / festive)
Aim for ~40 named combinations across these families.
